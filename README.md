# 花名册 - 花与人的收集游戏

一个简单的手机端网页应用，用于记录和查询花与人之间的对应关系。

## 功能

- **按花查询**：输入花名，查看谁拥有这种花
- **按人查询**：输入人名，查看这个人有什么花
- **添加记录**：新增"人-花"对应关系
- **数据管理**：支持添加/删除花、添加/删除人、添加/删除对应关系

## 技术栈

- 纯前端 HTML + CSS + JavaScript
- localStorage 本地存储
- 无需后端服务器

## 数据结构

```javascript
{
  flowers: ["玫瑰", "百合", ...],      // 花名列表
  people: ["小明", "小红", ...],       // 人名列表
  collections: [                      // 收藏关系
    { person: "小明", flower: "玫瑰" },
    { person: "小明", flower: "百合" },
    ...
  ]
}
```
